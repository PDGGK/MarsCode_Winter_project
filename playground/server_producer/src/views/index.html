1. Project

init [GET]

url

/project/init

params

名称 类型 描述
name string 应用名称
leader string 应用负责人
desc string 应用描述（非必填）

response

            {
              code: 0,
              data: {
                name: 'xxx',
                leader: 'xxx',
                desc: 'xxx'
              },
              msg: 'success'
            }

2. Log

report [POST/GET]

url

/log/report

params

名称 类型 描述
aid string 应用id
lid string 日志id
t string 上报时间
dat string 上报信息（json字符串）
b string 面包屑（json字符串）
url string 页面路径
lan string 浏览器语言
ua string 浏览器 userAgent
ttl string 页面标题

response

            {
              code: 0,
              data: null,
              msg: 'success'
            }

4. SourceMap

upload [POST]

url

/sourcemap/upload

params[FormData]

名称 类型 描述
file 文件流对象
dirname string 目录名称
filename string 文件名称
